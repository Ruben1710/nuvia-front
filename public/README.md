# Public Assets

## Logo
Place your NUVIA logo file here as `logo.png` (or update the path in `shared/ui/logo/logo.tsx`).

## Slider Images
For the hero slider, you can add images in the following paths:
- `/public/images/mugs.jpg` - For mugs slide
- `/public/images/tshirts.jpg` - For t-shirts slide
- `/public/images/badges.jpg` - For badges slide
- `/public/images/gifts.jpg` - For gifts slide
- `/public/images/souvenirs.jpg` - For souvenirs slide

Currently, the slider uses a gradient background with NUVIA text as a placeholder. Once you add images, they will automatically be used.
