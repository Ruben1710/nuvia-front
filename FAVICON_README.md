# Favicon Setup

## Для замены favicon:

1. **Поместите ваш файл `favicon.ico` в папку `app/`:**
   ```
   app/icon.ico
   ```

2. **Next.js App Router автоматически обработает файл:**
   - Создаст все необходимые размеры
   - Добавит правильные мета-теги
   - Будет работать во всех браузерах

## Альтернативные форматы:

Если у вас есть PNG файл, вы можете использовать:
- `app/icon.png` - основной favicon
- `app/apple-icon.png` - для Apple устройств (опционально)

## Текущая конфигурация:

- Favicon настроен в `app/layout.tsx` через metadata
- Файл должен быть назван `icon.ico` или `icon.png`
- Next.js автоматически обработает файл при сборке

## Проверка:

После замены файла:
1. Запустите `npm run build`
2. Проверьте в браузере: `http://localhost:3000`
3. Очистите кэш браузера (Ctrl+Shift+R / Cmd+Shift+R)
